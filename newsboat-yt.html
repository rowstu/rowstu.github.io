  1. Go to the YouTube channel page
  2. Right-click anywhere → View Page Source
  3. Select all (Ctrl+A) and copy (Ctrl+C)
  4. Paste into the box below

Enter a YouTube channel ID directly. You can find it in the channel's page URL or page source — it starts with UC and is 24 characters long.

Example: UCYO_jab_esuFRV4b17AJtAw

Usage

Paste a YouTube URL or @handle and this tool resolves the channel ID, then builds the RSS feed URL for Newsboat.

Add the generated line to your ~/.newsboat/urls file:

https://www.youtube.com/feeds/videos.xml?channel_id=UCxxxx "~Channel Name" "youtube"

Simplifying Expressions

Problem

Simplify the following expression:

  √65 + 8
  ─────────
  √65 - 8

Solution

Step 1: Rationalize the Denominator

To eliminate the square root in the denominator, multiply both numerator and denominator by the conjugate of the denominator (√65 + 8):

  √65 + 8     √65 + 8       (√65 + 8)²
  ───────  ×  ───────  =  ─────────────────────
  √65 - 8     √65 + 8     (√65 - 8)(√65 + 8)

Step 2: Simplify the Denominator

The denominator simplifies using the difference of squares formula:

(√65 - 8)(√65 + 8) = (√65)² - 8² = 65 - 64 = 1

Step 3: Simplify the Numerator

Expand the square of the numerator:

(√65 + 8)² = (√65)² + 2(√65)(8) + 8²

Simplify each term:

(√65)²65
2(√65)(8)16√65
8²64

Adding these together:

65 + 16√65 + 64 = 129 + 16√65

Final Answer

The simplified expression is:

129 + 16√65

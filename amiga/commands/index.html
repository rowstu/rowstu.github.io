Examples

Copying Files Example

To copy a file named example.txt to a directory called backup, you would use:

COPY example.txt backup/

Deleting a File Example

To delete a file named obsolete.txt, you would use:

DELETE obsolete.txt

Listing Contents of a Directory

To see everything in the documents directory, you could execute:

DIR documents/

Google Pixel 8a - Stock Firmware Debloat & Privacy Hardening

Conservative debloat and privacy guide for the Pixel 8a on stock firmware. The Pixel is already close to “stock Android” so this is really about stripping Google user apps, reducing telemetry, and tightening battery life.

Key Principles

  • Pixel is already stock AOSP - you’re removing Google’s own apps, not OEM bloat.
  • Use pm uninstall -k --user 0 to remove apps for your user only. The -k flag keeps data/cache.
  • Everything is recoverable via cmd package install-existing.
  • SafetyNet/Play Integrity stays intact, banking apps keep working, OTA updates still apply.

Prerequisites

Enable Developer options > USB debugging on the device.

adb devices
adb shell getprop ro.product.model

Make an Inventory First

# All packages
adb shell pm list packages > packages_all.txt

# System apps only
adb shell pm list packages -s > packages_system.txt

# Third-party (user installed) apps
adb shell pm list packages -3 > packages_thirdparty.txt

# Inspect a specific package
adb shell dumpsys package com.google.android.youtube | head -n 40

Core ADB Commands

Remove an app (recommended method)

adb shell pm uninstall -k --user 0 <package.name>

Disable an app (safer for uncertain packages)

adb shell pm disable-user --user 0 <package.name>

Restore a removed app

adb shell cmd package install-existing <package.name>

Re-enable a disabled app

adb shell pm enable <package.name>

DO NOT TOUCH - Red Line Packages

These will break core functionality if removed:

  • com.android.systemui - System UI
  • com.android.permissioncontroller - Permission system
  • com.google.android.gms - Play Services
  • com.google.android.gsf - Google Services Framework
  • com.android.phone - Telephony
  • com.android.providers.* - Content providers
  • com.android.settings - Settings app
  • com.android.vending - Play Store

Privacy & Telemetry Settings

On-Device Settings

Ad personalisation (Settings > Privacy & security > Ads):

  • Turn off Ad Topics, App-suggested ads, Ad measurement
  • Reset advertising ID

Activity tracking (Settings > Privacy & security > Activity controls):

  • Turn OFF Web & App Activity
  • Turn OFF Location History
  • Turn OFF YouTube History
  • Set auto-delete to 3 months

Usage & diagnostics (Settings > Privacy & security > Usage & diagnostics):

  • Turn OFF send usage and diagnostic data

Permissions cleanup (Settings > Privacy > Permission manager):

  • Location: set most apps to “While using” or “Don’t allow”
  • Microphone: remove for anything that doesn’t need it
  • Camera: same
  • Check: Nearby devices, Physical activity, Files & media

Private DNS (system-wide tracker blocking)

Settings > Network & Internet > Private DNS

Set to one of:

dns.adguard.com
family.cloudflare-dns.com

Blocks ad/telemetry domains with zero battery hit.

Lock Down Play Services Permissions

Settings > Privacy > Permission manager > Google Play services

Set ALL of these to Deny:

  • Location
  • Microphone
  • Camera
  • Physical activity
  • Files & media
  • Phone
  • Nearby devices

Play Services still works for most apps with these denied.

Restrict Play Services Battery

Settings > Apps > Google Play services > Battery > Restricted

Drops background drain by 20-40%. Notifications still work.

Battery Optimisations

On-Device Settings

Adaptive Battery (Settings > Battery > Adaptive preferences):

  • Enable Adaptive Battery
  • Enable Adaptive charging

Restrict background hogs (Settings > Battery > Battery usage):

  • Set high-drain apps to Restricted (social/media apps especially)

Disable always-listening features (Settings > System > Gestures):

  • Quick Tap: OFF
  • Lift to check phone: OFF
  • Now Playing: OFF (always-on music detection)

Location scanning (Settings > Location > Location services):

  • Wi-Fi scanning: OFF
  • Bluetooth scanning: OFF

Network-Level Blocking

Install RethinkDNS or NetGuard from F-Droid for surgical control.

Block these domains (safe tier):

google-analytics.com
app-measurement.com
firebaseinstallations.googleapis.com
firebaselogging.googleapis.com
doubleclick.net
ads.mopub.com

Allow these (required for app functionality):

mtalk.google.com          # push notifications
play.googleapis.com       # Play Store

Monitor What’s Still Phoning Home

adb shell dumpsys netstats | grep google

After debloat + firewall, you should see minimal traffic - mostly just Play Services for notification sync.

Recommended Sweet Spot

  • Stay stock Pixel (locked bootloader)
  • Debloat Google apps via ADB
  • Restrict Play Services permissions & battery
  • Private DNS + firewall (RethinkDNS)
  • Replace Google apps with F-Droid alternatives

This gives nearly GrapheneOS-level privacy without the pain.

If Going Further: microG

For custom ROMs (LineageOS etc.), microG is an open-source Play Services replacement:

  • Provides push notifications
  • Spoofs Play Services APIs
  • 90%+ app compatibility
  • Banking apps: mostly OK (some are picky)
  • Does NOT phone home to Google constantly

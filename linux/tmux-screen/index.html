Terminal Multiplexers: tmux and screen

Terminal multiplexers allow you to run multiple terminal sessions within a single window, detach from sessions and reattach later, and keep processes running after disconnecting.

tmux

Installation

# Debian/Ubuntu
sudo apt install tmux

# RHEL/CentOS/Fedora
sudo dnf install tmux

# Arch
sudo pacman -S tmux

Basic Commands

# Start a new session
tmux

# Start a named session
tmux new -s mysession

# List sessions
tmux ls

# Attach to last session
tmux attach

# Attach to named session
tmux attach -t mysession

# Kill a session
tmux kill-session -t mysession

# Kill all sessions
tmux kill-server

Configuration (~/.tmux.conf)

# Set prefix to Ctrl+a (like screen)
unbind C-b
set -g prefix C-a
bind C-a send-prefix

# Enable mouse support
set -g mouse on

# Start windows and panes at 1, not 0
set -g base-index 1
setw -g pane-base-index 1

# Renumber windows when one is closed
set -g renumber-windows on

# Increase history limit
set -g history-limit 10000

# Better split bindings
bind | split-window -h -c "#{pane_current_path}"
bind - split-window -v -c "#{pane_current_path}"

# Reload config
bind r source-file ~/.tmux.conf \; display "Config reloaded!"

# Vi-style copy mode
setw -g mode-keys vi

# Status bar
set -g status-style 'bg=#333333 fg=#ffffff'
set -g status-left '[#S] '
set -g status-right '%H:%M %d-%b-%y'

Reload config without restarting:

tmux source-file ~/.tmux.conf

GNU Screen

Installation

# Debian/Ubuntu
sudo apt install screen

# RHEL/CentOS/Fedora
sudo dnf install screen

# Arch
sudo pacman -S screen

Basic Commands

# Start a new session
screen

# Start a named session
screen -S mysession

# List sessions
screen -ls

# Attach to a session
screen -r mysession

# Attach to a detached session (or detach and reattach)
screen -d -r mysession

# Detach a running session remotely
screen -d mysession

# Kill a detached session
screen -X -S mysession quit

Configuration (~/.screenrc)

# Disable startup message
startup_message off

# Use visual bell
vbell on

# Large scrollback buffer
defscrollback 10000

# Status line
hardstatus alwayslastline
hardstatus string '%{= kG}[%{G}%H%{g}][%= %{= kw}%?%-Lw%?%{r}(%{W}%n*%f%t%?(%u)%?%{r})%{w}%?%+Lw%?%?%= %{g}][%{B}%Y-%m-%d %{W}%c%{g}]'

# Default windows
screen -t shell 0
screen -t logs 1

# Enable mouse scrolling
termcapinfo xterm* ti@:te@

# Detach on hangup
autodetach on

Common Use Cases

Keep a process running after SSH disconnect

# Start tmux/screen, run your command, then detach
tmux new -s backup
./long-running-backup.sh
# Press Ctrl+b d to detach

# Later, reattach
tmux attach -t backup

Share a terminal session

# User 1: Create a shared session
tmux new -s shared

# User 2: Attach to the same session
tmux attach -t shared

Run multiple services in development

# Create windows for different services
tmux new -s dev
# Ctrl+b c (new window for each service)
# Window 0: frontend
# Window 1: backend
# Window 2: database logs

IRC – Internet Relay Chat Guide

Home / Browse/ Linuxirc

IRC (Internet Relay Chat) is a real-time text messaging protocol from 1988 that is still widely used by open source projects, Linux communities, and tech groups. It’s lightweight, no-frills, and runs in a terminal.

Quick Start

# Install a client
sudo apt install irssi          # Debian/Ubuntu
sudo pacman -S irssi            # Arch
sudo dnf install irssi          # Fedora

# Or use weechat (more modern)
sudo apt install weechat
sudo pacman -S weechat

Connect to a network:

irssi -c irc.libera.chat -n YourNick

Or in WeeChat:

weechat
/server add libera irc.libera.chat/6697 -ssl
/connect libera

Core Concepts

TermMeaning
NetworkAn IRC network (e.g. Libera Chat, OFTC, EFnet). Each is independent
ServerA host on the network you connect to (e.g. irc.libera.chat)
ChannelA chat room, always prefixed with # (e.g. #linux)
NickYour display name on the network
NickServA service that lets you register and protect your nick
ChanServA service that manages channel registration and permissions
Op (@)Channel operator — can kick, ban, set topic
Voice (+)A user given voice in a moderated channel
CTCPClient-to-client protocol (VERSION, PING, etc.)

Everyday Commands

These work in virtually every IRC client.

Connecting & Identity

/connect irc.libera.chat        Connect to a server
/disconnect                     Disconnect from current server
/nick MyNewNick                 Change your nickname
/quit Goodbye!                  Quit IRC with an optional message

Channels

/join #channel                  Join a channel
/part #channel                  Leave a channel
/part #channel See you later    Leave with a message
/list                           List all channels (can be huge — use /list *linux* to filter)
/topic #channel New topic       Set channel topic (if you have permission)
/names #channel                 List users in a channel

Messaging

/msg NickName Hello!            Send a private message
/notice NickName Hello!         Send a notice (less intrusive than msg)
/me does something              Action message — shows as "* YourNick does something"
/away Gone for lunch            Mark yourself as away
/away                           Remove away status

User Information

/whois NickName                 Look up info about a user
/who #channel                   List users in a channel with details
/userhost NickName              Quick host lookup

Channel Management (Ops)

/kick #channel NickName reason  Kick a user
/ban #channel *!*@hostname      Ban a host pattern
/mode #channel +o NickName      Give someone operator status
/mode #channel +v NickName      Give someone voice
/mode #channel +m               Set channel to moderated (only voiced/ops can talk)
/mode #channel +i               Set channel to invite-only
/mode #channel +t               Only ops can change topic

NickServ – Registering Your Nick

Most networks let you register a nick so nobody else can use it.

/msg NickServ REGISTER password email@example.com     Register your nick
/msg NickServ IDENTIFY password                        Log in to your nick
/msg NickServ SET ENFORCE ON                           Auto-kick imposters after 30s
/msg NickServ INFO NickName                            Check if a nick is registered
/msg NickServ GHOST NickName password                  Kill a ghost session using your nick

Irssi Quick Reference

Irssi is the most common terminal IRC client on Linux.

Window Management

Alt+1 to Alt+9          Switch to window 1-9
Alt+0                   Switch to window 10
/window N               Switch to window N (for higher numbers)
/window close           Close current window
/window list            List all windows
Ctrl+N / Ctrl+P         Next / previous window
Page Up / Page Down     Scroll history
/scrollback clear       Clear scrollback buffer

Configuration

# Config file location
~/.irssi/config

# Useful settings
/set nick YourNick
/set real_name Your Name
/set user_name username
/set theme default

# Auto-connect on startup
/network add libera
/server add -auto -network libera irc.libera.chat 6697 -ssl
/channel add -auto #linux libera

# Save your config
/save

Logging

/set autolog ON                 Enable auto-logging
/set autolog_path ~/irclogs/%Y/$tag/$0.%m-%d.log

Using IRC Over TLS/SSL

Always use TLS when connecting. Most networks support it on port 6697.

# irssi
/connect -tls irc.libera.chat 6697

# weechat — SSL is set per-server
/server add libera irc.libera.chat/6697 -ssl
/set irc.server.libera.ssl on
/set irc.server.libera.ssl_verify on

IRC Etiquette

  • Don’t ask to ask — just ask your question directly
  • Be patient — people are in different timezones, wait at least 15-20 minutes
  • Use a pastebin for anything longer than 3 lines (e.g. 0x0.st, ix.io)
  • Read the channel topic — it often has rules and useful links
  • Don’t PM people without asking in the channel first
  • Use your nick consistently so people recognise you

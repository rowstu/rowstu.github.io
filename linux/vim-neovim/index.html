Vim and Neovim Keyboard Shortcuts

A reference guide for common Vim and Neovim keyboard shortcuts. These work in both editors unless noted otherwise.

Modes

KeyAction
EscReturn to Normal mode
iInsert mode (before cursor)
aInsert mode (after cursor)
IInsert mode (start of line)
AInsert mode (end of line)
oInsert mode (new line below)
OInsert mode (new line above)
vVisual mode (character)
VVisual mode (line)
Ctrl+vVisual block mode
RReplace mode
:Command-line mode

Navigation

Basic Movement

KeyAction
hLeft
jDown
kUp
lRight
gjDown (visual line, for wrapped text)
gkUp (visual line, for wrapped text)

Word Movement

KeyAction
wNext word (start)
WNext WORD (whitespace-delimited)
eNext word (end)
ENext WORD (end)
bPrevious word (start)
BPrevious WORD
gePrevious word (end)

Line Movement

KeyAction
0Start of line
^First non-blank character
$End of line
g_Last non-blank character
f{char}Find next {char} on line
F{char}Find previous {char} on line
t{char}Till next {char} (cursor before)
T{char}Till previous {char}
;Repeat last f/F/t/T
,Repeat last f/F/t/T (reverse)

Screen Movement

KeyAction
HTop of screen
MMiddle of screen
LBottom of screen
Ctrl+fPage down
Ctrl+bPage up
Ctrl+dHalf page down
Ctrl+uHalf page up
Ctrl+eScroll down one line
Ctrl+yScroll up one line
zzCentre cursor on screen
ztCursor to top of screen
zbCursor to bottom of screen

Document Movement

KeyAction
ggStart of file
GEnd of file
{n}GGo to line n
:{n}Go to line n
%Jump to matching bracket
{Previous paragraph
}Next paragraph
[[Previous section
]]Next section

Marks

KeyAction
m{a-z}Set local mark
m{A-Z}Set global mark
`{mark}Jump to mark (exact position)
'{mark}Jump to mark (line start)
``Jump to position before last jump
`.Jump to last edit
:marksList all marks

Editing

Basic Editing

KeyAction
xDelete character
XDelete character before cursor
r{char}Replace character
sSubstitute character (delete + insert)
SSubstitute line
JJoin line below
gJJoin line below (no space)
uUndo
Ctrl+rRedo
.Repeat last change
~Toggle case
g~{motion}Toggle case over motion
gu{motion}Lowercase over motion
gU{motion}Uppercase over motion

Delete (cut)

KeyAction
d{motion}Delete over motion
ddDelete line
DDelete to end of line
dwDelete word
diwDelete inner word
dawDelete a word (with space)
di"Delete inside quotes
da"Delete around quotes
di(Delete inside parentheses
dipDelete inner paragraph
ditDelete inside tag

Change (delete + insert)

KeyAction
c{motion}Change over motion
ccChange line
CChange to end of line
cwChange word
ciwChange inner word
ci"Change inside quotes
ci(Change inside parentheses
citChange inside tag

Yank (copy)

KeyAction
y{motion}Yank over motion
yyYank line
YYank line (or to end with :nnoremap Y y$)
ywYank word
yiwYank inner word
yi"Yank inside quotes

Put (paste)

KeyAction
pPut after cursor
PPut before cursor
gpPut after and move cursor after
gPPut before and move cursor after
]pPut with adjusted indent

Registers

KeyAction
"{reg}Use register for next delete/yank/put
"ay{motion}Yank to register a
"apPut from register a
"+yYank to system clipboard
"+pPut from system clipboard
"0pPut from yank register
:regShow registers

Search and Replace

Search

KeyAction
/{pattern}Search forward
?{pattern}Search backward
nNext match
NPrevious match
*Search word under cursor (forward)
#Search word under cursor (backward)
g*Search partial word (forward)
g#Search partial word (backward)
:nohClear search highlighting

Replace

KeyAction
:%s/old/new/gReplace all in file
:%s/old/new/gcReplace all with confirmation
:s/old/new/gReplace all in line
:'<,'>s/old/new/gReplace in visual selection
:5,10s/old/new/gReplace in lines 5-10
&Repeat last substitution
g&Repeat last substitution globally

Windows and Tabs

Windows

KeyAction
Ctrl+w sSplit horizontally
Ctrl+w vSplit vertically
Ctrl+w wSwitch windows
Ctrl+w h/j/k/lMove to window
Ctrl+w H/J/K/LMove window
Ctrl+w =Equal window sizes
Ctrl+w _Maximise height
`Ctrl+w`
Ctrl+w +/-Increase/decrease height
Ctrl+w >/<Increase/decrease width
Ctrl+w qClose window
Ctrl+w oClose all other windows
Ctrl+w TMove window to new tab

Tabs

KeyAction
:tabnewNew tab
:tabnew {file}Open file in new tab
gtNext tab
gTPrevious tab
{n}gtGo to tab n
:tabcloseClose tab
:tabonlyClose all other tabs
:tabmove {n}Move tab to position n

Buffers

KeyAction
:e {file}Edit file
:lsList buffers
:b {n}Switch to buffer n
:bnNext buffer
:bpPrevious buffer
:bdDelete buffer (close file)
:b#Switch to alternate buffer
Ctrl+^Switch to alternate buffer

Useful Commands

CommandAction
:wSave
:w {file}Save as
:qQuit
:q!Quit without saving
:wq or :x or ZZSave and quit
ZQQuit without saving
:e!Revert to saved
:!{cmd}Run shell command
:r !{cmd}Insert command output
:r {file}Insert file contents
:%!{cmd}Filter buffer through command
:earlier {time}Undo to earlier state (e.g., 10m)
:later {time}Redo to later state
:set pasteEnable paste mode
:set nopasteDisable paste mode
:set numberShow line numbers
:set relativenumberShow relative line numbers
:set listShow hidden characters
:retabConvert tabs to spaces
:%y+Yank entire file to clipboard
:sortSort lines
:sort uSort and remove duplicates

Macros

KeyAction
q{a-z}Start recording macro to register
qStop recording
@{a-z}Play macro
@@Replay last macro
{n}@{a-z}Play macro n times

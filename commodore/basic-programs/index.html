Commodore BASIC Programs to Try

Home / Browse/ Commodorebasic Programs

The C64 boots straight into Commodore BASIC V2. It’s limited compared to later BASICs (no commands for graphics or sound — you poke memory directly), but that’s part of the charm. Here are some programs to type in, from trivial to genuinely useful.

The Essentials

Hello World

10 PRINT "HELLO WORLD"
20 GOTO 10
RUN

The simplest infinite loop. Press RUN/STOP to break out.

The Maze

The most famous C64 one-liner. Prints a random maze using PETSCII diagonal characters.

10 PRINT CHR$(205.5+RND(1)); : GOTO 10

CHR$(205) is / and CHR$(206) is \ in the PETSCII character set. The RND(1) picks one at random.

Random Colour Flasher

10 POKE 53280,RND(1)*16
20 POKE 53281,RND(1)*16
30 FOR T=1 TO 100:NEXT
40 GOTO 10

53280 is the border colour register, 53281 is the background. Values 0-15.

Graphics

Plot a Pixel (Bitmap Mode)

This sets up hi-res bitmap mode and plots a single pixel. The C64’s bitmap is at $2000 (8192) by default.

10 REM ENABLE BITMAP MODE
20 POKE 53265, PEEK(53265) OR 32
30 POKE 53272, PEEK(53272) OR 8
40 REM CLEAR BITMAP (8KB)
50 FOR I=8192 TO 16191:POKE I,0:NEXT
60 REM SET COLOUR (WHITE ON BLACK)
70 FOR I=1024 TO 2023:POKE I,16:NEXT
80 REM PLOT A PIXEL AT X=160, Y=100
90 X=160:Y=100
100 BASE=8192+INT(Y/8)*320+INT(X/8)*8+(Y AND 7)
110 BIT=7-(X AND 7)
120 POKE BASE, PEEK(BASE) OR (2^BIT)
130 GET A$:IF A$="" THEN 130
140 REM RESTORE TEXT MODE
150 POKE 53265, PEEK(53265) AND 223

Slow to clear (line 50), but it works. Press any key to exit.

Sine Wave

Draws a sine wave using character plotting.

10 PRINT CHR$(147)
20 FOR X=0 TO 39
30 Y=INT(12+11*SIN(X/6.28*2*3.14159))
40 POKE 1024+Y*40+X,81
50 POKE 55296+Y*40+X,1
60 NEXT

Screen memory starts at 1024, colour RAM at 55296. Character 81 is a filled circle.

Sound

Simple Beep

10 POKE 54296,15 : REM VOLUME MAX
20 POKE 54277,9  : REM ATTACK/DECAY
30 POKE 54278,0  : REM SUSTAIN/RELEASE
40 POKE 54273,28 : REM FREQ HI
50 POKE 54272,49 : REM FREQ LO
60 POKE 54276,17 : REM GATE ON, TRIANGLE WAVE
70 FOR T=1 TO 500:NEXT
80 POKE 54276,16 : REM GATE OFF
90 POKE 54296,0  : REM VOLUME OFF

The SID registers start at 54272 ($D400). Voice 1 is registers 0-6 from that base.

Three-Voice Chord

Play a C major chord using all three SID voices.

10 S=54272
20 POKE S+24,15        : REM VOLUME
30 REM VOICE 1 - C (NOTE)
40 POKE S+1,17:POKE S+0,37
50 POKE S+5,9:POKE S+6,0
60 POKE S+4,17
70 REM VOICE 2 - E
80 POKE S+8,21:POKE S+7,154
90 POKE S+12,9:POKE S+13,0
100 POKE S+11,17
110 REM VOICE 3 - G
120 POKE S+15,25:POKE S+14,30
130 POKE S+19,9:POKE S+20,0
140 POKE S+18,17
150 FOR T=1 TO 2000:NEXT
160 POKE S+4,16:POKE S+11,16:POKE S+18,16
170 POKE S+24,0

Music Player (Simple)

Plays a scale using a DATA statement for note frequencies.

10 S=54272
20 POKE S+24,15:POKE S+5,9:POKE S+6,0
30 READ HI,LO:IF HI=-1 THEN 70
40 POKE S+1,HI:POKE S+0,LO
50 POKE S+4,17:FOR T=1 TO 200:NEXT
60 POKE S+4,16:FOR T=1 TO 50:NEXT:GOTO 30
70 POKE S+24,0:END
100 DATA 17,37, 19,63, 21,154, 23,59
110 DATA 25,30, 28,49, 31,165, 34,75
120 DATA -1,0

Useful Utilities

Memory Viewer

A simple hex dump of any memory region.

10 INPUT "START ADDRESS";A
20 FOR ROW=0 TO 7
30 PRINT RIGHT$("0000"+MID$(STR$(A),2),5);": ";
40 FOR COL=0 TO 7
50 V=PEEK(A):A=A+1
60 H$=HEX$(V):IF LEN(H$)<2 THEN H$="0"+H$
70 PRINT H$;" ";
80 NEXT COL
90 PRINT
100 NEXT ROW
110 PRINT:GOTO 20

Note: HEX$() is not available in stock BASIC V2. For stock BASIC, replace line 60:

60 H=INT(V/16):L=V-H*16
65 PRINT MID$("0123456789ABCDEF",H+1,1);MID$("0123456789ABCDEF",L+1,1);" ";

Disk Directory Viewer

10 OPEN 1,8,0,"$"
20 GET#1,A$,B$
30 GET#1,A$,B$
40 IF ST THEN 70
50 GET#1,A$,B$
55 IF A$="" THEN A$=CHR$(0)
56 IF B$="" THEN B$=CHR$(0)
60 PRINT ASC(A$)+ASC(B$)*256;
65 GET#1,A$:IF A$<>CHR$(0) THEN PRINT A$;:GOTO 65
66 PRINT:GOTO 40
70 CLOSE 1

Joystick Tester

Read joystick port 2 and display directions and fire button.

10 PRINT CHR$(147);"JOYSTICK TESTER"
20 PRINT "MOVE STICK AND PRESS FIRE"
30 J=PEEK(56320)
40 PRINT CHR$(19)
50 PRINT:PRINT
60 IF (J AND 1)=0 THEN PRINT "UP   ";:GOTO 70
65 PRINT "     ";
70 IF (J AND 2)=0 THEN PRINT "DOWN ";:GOTO 80
75 PRINT "     ";
80 IF (J AND 4)=0 THEN PRINT "LEFT ";:GOTO 90
85 PRINT "     ";
90 IF (J AND 8)=0 THEN PRINT "RIGHT";:GOTO 100
95 PRINT "     ";
100 PRINT
110 IF (J AND 16)=0 THEN PRINT "FIRE!  ";:GOTO 120
115 PRINT "       ";
120 GOTO 30

Joystick port 2 is at 56320 ($DC00). Bits are active-low (0 = pressed).

Tips

  • Always number lines in increments of 10 so you can insert lines between them later
  • CHR$(147) clears the screen, CHR$(19) moves cursor to home
  • Use POKE 53265,11 to blank the screen for faster BASIC execution (the VIC chip steals fewer cycles). POKE 53265,27 to restore
  • SAVE before RUN — there’s no undo and a crash loses everything
  • The C64 screen editor lets you cursor up to any line and press RETURN to re-enter it — you can edit programs by overtyping lines directly on screen
  • To delete a line, type just the line number and press RETURN

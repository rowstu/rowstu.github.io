JiffyDOS – Speed Up Your C64 Disk Access

Home / Browse/ Commodorejiffydos

JiffyDOS is a replacement ROM for the C64 (and 1541/1571/1581 drives) that dramatically speeds up disk access and adds a set of useful DOS wedge commands. It was originally sold in the late 1980s and is still available as a licensed product.

What It Does

The standard C64-to-1541 serial bus transfer rate is around 300 bytes/second — painfully slow. JiffyDOS replaces the serial protocol with a faster byte transfer routine, achieving roughly 10x speed improvement.

OperationStock speedJiffyDOS speed
Load a 200-block program~80 seconds~8 seconds
Directory listing~8 seconds<1 second
Sequential file accessSlow~10x faster
Format a diskSameSame (hardware limited)

Both the computer and the drive need JiffyDOS ROMs for the speed improvement. If only one side has it, the system falls back to standard speed.

Installation

JiffyDOS requires replacing ROM chips. This is a physical hardware mod.

C64 Side (Kernal ROM)

  1. The C64 Kernal ROM is at U4 (breadbin) or U4 (C64C) — a 28-pin chip
  2. Remove the original 901227-03 Kernal ROM
  3. Install the JiffyDOS Kernal ROM in its place
  4. If the socket isn’t present, you’ll need to desolder and add a 28-pin DIP socket first
  5. Many people install a switch to toggle between the stock Kernal and JiffyDOS — wire pin 27 (A14) to a SPDT switch to toggle between the two 8KB banks

Drive Side (DOS ROM)

  1. Open the 1541 drive
  2. Locate the DOS ROM (typically at UB4, a 16KB ROM)
  3. Replace with the JiffyDOS drive ROM
  4. Again, a switch is recommended for compatibility

Switchless Options

  • Switchless Kernal boards (like the SKS64) let you hold RESTORE during power-on to cycle between multiple kernals
  • The 1541 Ultimate II+ has JiffyDOS support built-in if you supply your own licensed ROM image

JiffyDOS Commands

JiffyDOS replaces the standard Kernal and adds a DOS wedge — shortcut commands using the @ symbol.

Directory and File Operations

@                     Display disk status / error channel
@$                    List directory
@$:pattern*           List files matching pattern
@S:filename           Scratch (delete) a file
@R:newname=oldname    Rename a file
@C:dest=source        Copy a file on the same disk
@N:diskname,id        Format (new) a disk
@I                    Initialise drive (re-read BAM)
@V                    Validate (fix) the disk's BAM

Loading and Saving

/filename             Load a PRG file (equivalent to LOAD"filename",8,1)
%filename             Load and run a PRG file
↑filename             Load BASIC (no ,1 — loads to BASIC start)
←filename             Save a file

The / key shortcut for loading is the single biggest quality-of-life improvement. No more typing LOAD"FILENAME",8,1 and then RUN.

Device Selection

@#8                   Select device 8 (default drive)
@#9                   Select device 9
@#10                  Select device 10

Once selected, all @ commands and / loads go to that device.

Function Keys

JiffyDOS maps the function keys to common operations:

KeyAction
F1List directory (@$)
F2Display disk status (@)
F3Scratch a file (prompts for name)
F4Load file under cursor from directory listing
F5Load + RUN file under cursor
F6Toggle device 8/9
F7Not assigned by default
F8Not assigned by default

The F4/F5 cursor load feature is extremely useful: list the directory with F1, move the cursor to a filename, press F5, and it loads and runs. No typing required.

Compatibility Notes

  • JiffyDOS works with the 1541, 1571, and 1581 drives (each needs its own ROM)
  • SD2IEC devices do NOT benefit from JiffyDOS speed — they use their own fast protocol. JiffyDOS wedge commands (@$, /, etc.) still work though
  • Pi1541 fully supports JiffyDOS when the C64 has the Kernal installed
  • 1541 Ultimate II+ supports JiffyDOS natively
  • A small number of games and demos are incompatible with JiffyDOS because they replace the Kernal entirely or use custom loaders that conflict. Use the switch to flip back to stock for these
  • GEOS works with JiffyDOS and benefits from the speed improvement

Where to Get It

JiffyDOS is a licensed product. Legitimate sources:

  • Retro Innovations (store.go4retro.com) — sells licensed JiffyDOS ROM chips and kits for C64, C128, 1541, 1571, 1581
  • The ROM images are also available for use with the 1541 Ultimate II+ (you need to provide proof of licence)

Tips

  • Always install with a switch. Some software won’t work with JiffyDOS active, and you’ll want to flip back to stock
  • If you have multiple drives, JiffyDOS them all — the speed improvement only works when both sides have the ROM
  • The @$ command is your best friend. Use it constantly
  • Get comfortable with the F5 cursor-load workflow: F1 to list, cursor to file, F5 to run. It’s faster than anything else
  • JiffyDOS pairs perfectly with a Pi1541 — you get the fast protocol AND cycle-exact compatibility

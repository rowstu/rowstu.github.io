Google Dorking

Google advanced search operators to surface hidden data.

site:example.com                    # restrict to domain
filetype:pdf                        # specific file type
intitle:"index of"                  # directory listings
inurl:admin                         # URL contains string
"@gmail.com" site:pastebin.com      # find leaked emails
cache:example.com                   # cached version of page
link:example.com                    # pages linking to site

Combine operators:

site:linkedin.com intitle:"at CompanyName"
filetype:xls site:gov.uk password
inurl:"/wp-content/uploads/" filetype:pdf confidential

Email Header Analysis

Email headers reveal routing, originating IP, mail client, and spam indicators.

# Extract received headers in order (bottom = origin)
# Look for:
# - X-Originating-IP
# - X-Mailer (email client)
# - Received: from (SMTP hops)
# - SPF / DKIM pass/fail

Useful tools: MXToolbox Header Analyzer, Google Admin Toolbox

Username Enumeration

# Sherlock - check username across platforms
python3 sherlock username

# Maigret - detailed report with 2000+ sites
python3 -m maigret username --pdf

# WhatsMyName via CLI
curl -s "https://whatsmyname.app/api/username" -d '{"username":"target"}'

Reverse Image Search Workflow

  1. Upload or paste image URL to Google Images, Yandex, TinEye
  2. Yandex is most effective for faces — try it first
  3. Check PimEyes for face recognition against public web
  4. Analyse EXIF data with ExifTool — look for GPS, camera model, timestamps
  5. Cross-reference GPS coords in Google Maps street view
exiftool image.jpg | grep -E "GPS|Date|Camera|Make|Model"

Domain / IP Footprinting

# Passive DNS history
host -t NS target.com
whois target.com

# Subdomain enumeration (passive)
# Use: DNSDumpster, SecurityTrails, crt.sh

# Certificate transparency logs
curl "https://crt.sh/?q=%.target.com&output=json" | jq '.[].name_value' | sort -u

# if the above fails
curl "https://crt.sh/?q=%.target.com&output=json" | jq '.[] | .name_value' | sort -u


# Shodan via CLI
shodan host 1.2.3.4
shodan search "hostname:target.com"

Social Media Account Pivoting

1. Find username → check across platforms (Sherlock/Maigret)
2. Profile photo → reverse image search → find other accounts
3. Email from profile → Holehe to find other registrations
4. Friends/followers list → build social graph
5. Post timestamps → infer timezone and routine
6. Tagged locations → cross-reference with mapping tools

OpSec When Doing OSINT

  • Use a VPN or Tor — your IP may be logged by target sites
  • Use a dedicated browser profile with no personal cookies/history
  • Use sock puppet accounts rather than your real identity
  • Be aware of rate limiting — stagger queries to avoid flagging
  • Never interact with the target — read-only observation only
  • Document everything with screenshots and timestamps

OSINT Tools Reference

Command-Line Tools

Sherlock

Hunt down social media accounts by username across 300+ platforms.

# Install
pip install sherlock-project

# Basic usage
sherlock username

# Output to file
sherlock username --output results.txt

# Check specific sites
sherlock username --site Twitter --site Instagram

Maigret

Username OSINT across 2000+ sites with detailed reports.

# Install
pip install maigret

# Run against username
maigret username

# Generate PDF report
maigret username --pdf

# JSON output
maigret username --json report.json

Holehe

Check whether an email address is registered on 120+ websites.

# Install
pip install holehe

# Basic usage
holehe email@example.com

# CSV output
holehe email@example.com --csv

PhoneInfoga

International phone number OSINT and carrier lookup.

# Install via Docker
docker pull sundowndev/phoneinfoga
docker run --rm sundowndev/phoneinfoga scan -n "+447911123456"

# Or via binary
phoneinfoga scan -n "+447911123456"
phoneinfoga serve  # Web UI on localhost:5000

theHarvester

Gather emails, subdomains, IPs, and URLs from public sources.

# Install
pip install theHarvester

# Search by domain
theHarvester -d target.com -b google,bing,linkedin

# Available sources: google, bing, linkedin, twitter, hunter, shodan, etc.
theHarvester -d target.com -b all -f output

# Limit results
theHarvester -d target.com -b google -l 200

SpiderFoot

Automated OSINT collection with 200+ modules.

# Install
pip install spiderfoot

# CLI scan
spiderfoot -s target.com -t INTERNET_NAME -m sfp_dnsresolve,sfp_whois

# Web UI
spiderfoot -l 127.0.0.1:5001

# Scan types:
# -t INTERNET_NAME  → domain
# -t IP_ADDRESS     → IP
# -t EMAILADDR      → email
# -t USERNAME       → username

ExifTool

Extract and edit metadata from files.

# Install (Arch)
sudo pacman -S perl-image-exiftool

# Show all metadata
exiftool image.jpg

# Extract GPS
exiftool -GPS* image.jpg

# Strip all metadata
exiftool -all= image.jpg

# Batch process
exiftool -r -GPS* /path/to/folder/

# Show specific tags
exiftool -Make -Model -DateTimeOriginal image.jpg

Recon-ng

Modular web reconnaissance framework.

# Install
pip install recon-ng

# Start
recon-ng

# Key commands
marketplace install all          # install modules
workspaces create target_name
db insert domains                # add target domain
modules load recon/domains-hosts/google_site_web
run
show hosts

Maltego (GUI)

Visual link analysis and pivot tool. Free Community Edition available.

Download from maltego.com

Key transforms:

  • Domain → IP, DNS records, MX, subdomains
  • Email → social profiles, breach data
  • Person → LinkedIn, Twitter, phone

Online Platforms

Shodan

Internet-connected device search engine.

# Useful Shodan filters
port:22 country:GB                          # SSH servers in UK
product:"Apache httpd" version:"2.4"       # specific web server version
hostname:target.com                         # by hostname
org:"Company Name"                          # by organisation
http.favicon.hash:116323821                 # by favicon hash
ssl.cert.subject.cn:target.com             # by TLS certificate

Censys

Certificate and service discovery.

# Search syntax
services.port: 22 AND location.country: GB
parsed.names: target.com                    # TLS cert domain
autonomous_system.name: "AS Name"

VirusTotal

# Search operators
domain:target.com
ip:1.2.3.4
entity:filename.exe
tag:signed AND tag:packed

Certificate Transparency

# Query crt.sh for subdomains
curl -s "https://crt.sh/?q=%.target.com&output=json" \
  | jq -r '.[].name_value' \
  | sed 's/\*\.//g' \
  | sort -u

# Amass passive enumeration
amass enum -passive -d target.com

WHOIS & DNS

# WHOIS
whois target.com
whois 1.2.3.4

# DNS records
dig target.com ANY
dig +short MX target.com
dig +short NS target.com

# Reverse DNS
dig -x 1.2.3.4

# Zone transfer attempt (educational)
dig axfr @ns1.target.com target.com

# Historical DNS via CLI (requires API key)
# SecurityTrails: https://securitytrails.com/app/api
curl -H "APIKEY: YOUR_KEY" "https://api.securitytrails.com/v1/domain/target.com/dns/A"
